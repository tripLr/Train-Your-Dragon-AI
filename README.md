# Skills

Skills are small, focused instruction sets that teach the AI how to work with you in a specific domain.

## Current Skills (Public / General)

- `personal-assistant` — Day-to-day support (to-dos, reminders, light organization)
- `research-skill` — Structured research that moves from evidence → personal understanding → multiple human lenses

## Private / Domain Skills

Course-specific and highly personal skills (for example Warrior’s Way study systems) stay private and are not part of the public project.
