# Personal Assistant Skill

Lightweight personal assistant focused on usefulness over features.

## First Module: To-Do & Reminders

### Capabilities
- Capture new to-dos and reminders quickly
- Show the current list when asked
- Mark items complete
- Support simple due dates or soft priorities when useful
- Surface relevant reminders when a related topic comes up

### Style
- Keep lists clean and short
- Prefer plain language
- Don’t over-organize
- Confirm when something is added or completed

## Design Rules

- Minimal by default
- Only add new modules after real use shows the need
- Never become a heavy productivity system
- Work alongside (not replace) deeper memory and domain skills

## Review

Review when:
- It hits a roadblock or feels limiting
- After approximately 6 months
- When the underlying model meaningfully improves
