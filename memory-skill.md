# Memory Skill

## Purpose

Support a durable, versioned personal memory system that favors growth, agency, and truth-seeking over static identity claims or noisy chat history.

This skill teaches both the human and the AI how to keep memory useful across months and model changes.

## Core Practices

- One clear fact or preference per entry
- Always include a date
- Prefer replace over stacking contradictory statements
- Hard-delete what is no longer true
- Keep the main file short enough that it stays useful

## Recommended Structure

1. **Memory Structure, Versioning & Intake Rules**  
   Short description of how the file is governed.

2. **Self Reflection & Versioned Reasoning**  
   Dated versions of deep processing work.  
   These remain open to ongoing review.  
   They are not fixed identity traits.

3. **Lightweight main sections**  
   - Who This User Is (current facts only)  
   - Core Interests (grouped by subject)  
   - Key Life Events (major milestones only)  
   Use short pointers to versioned material when needed.

## Intake Process (use every time)

When new information appears, ask:

- Is this new or similar to something already present?
- What subject does it belong to?
- For any belief or emotional pattern: separate *belief / judgment / feeling / fact* using who / what / when / where / why logic.
- Does this belong in a short main-section entry or in a dated versioned reflection?

Only store durable facts and preferences. Ephemeral states, third-party details, and temporary session context do not belong.

## AI Behavior

- Treat the memory file as the durable external system.
- When the user shares personal information, apply the intake process above.
- When deep self-reflection or belief examination occurs, offer to capture it as a dated version rather than as a permanent claim.
- Surface review triggers when a section feels stale or limiting.
- Never turn versioned processing work into fixed identity labels.

## Review Triggers

Review the memory system (and this skill) when:
- It hits a roadblock or feels limiting
- Approximately 6 months have passed
- The underlying model meaningfully improves
